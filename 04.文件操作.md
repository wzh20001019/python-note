#### 1. 文件读取

```python
# 打开文件 open(name, mode, encoding)
# name: 文件路径
# mode: 设置文件打开模式  r: 只读,  w: 写入,  a: 追加
# encoding: 编码格式 UTF-8
file = open('./1.txt', 'r', encoding="UTF-8")

# 读取文件 read()
file.read()  # 读取全不内容

file.readline()  # 读取一行内容
file.readlines()  # 读取全不内容, 并封装到列表中

# 关闭文件
file.close()


# with open()
with open() as file:
    # 文件操作, 会自动关闭文件
    file.read()
```

#### 2. 文件写入

```python
file = open('./1.txt', 'w', encoding="UTF-8")

# 写入文件 写入到内存中(避免频繁操作硬盘)
file.write("写入会覆盖原来的内容")

# 写入到硬盘文件
file.flush()

# 关闭文件 自带 flush功能能
file.close()
```

#### 3. 追加内容

```python
file = open('./1.txt', 'a', encoding="UTF-8")

# 写入文件
file.write("\n追加内容")

# 写入到硬盘文件
file.flush()

# 关闭文件 自带 flush功能能
file.close()
```
